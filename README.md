# Nodebeats
Don't let the beat drop

## Get started
1. `npm install`
2. `cp .env.example .env`
3. `npm start`. Optionally install `pm2` and use the ecosystem.json config.

## Testing
1. Install `mocha` globally `npm install mocha -g`
2. `npm test`
